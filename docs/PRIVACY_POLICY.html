Introduction

Online Picket Line ("OPL", "we", "our", or "the extension") is a browser extension that informs users about active labor actions (strikes, boycotts, etc.) when they visit websites associated with those actions. This privacy policy explains what data we collect, how we use it, and your rights regarding your information.

Our Privacy Commitment

We do not collect, store, or transmit your browsing history or personal information.

The extension is designed with privacy-first principles:

  • Uses modern declarativeNetRequest API that cannot access page content by design
  • All URL matching happens locally in your browser
  • No tracking, no analytics, no user profiling
  • Open source code available for review

How the Extension Works

Block Mode (Default)

When block mode is enabled, the browser's declarativeNetRequest API matches URLs against rules at the browser level. No JavaScript execution occurs unless there's a match. If a match is found, the browser redirects to our informational page with cached details - no URL is transmitted anywhere.

Technical Privacy Guarantee: The declarativeNetRequest API cannot access page content by architectural design. Chrome engineered this API specifically for privacy.

Banner Mode (Non-Intrusive)

When banner mode is enabled, a lightweight script (~5KB) checks the current URL against locally cached labor action patterns. All matching happens entirely in your browser. If a match is found, an informational banner is displayed. No network requests or data transmission occurs.

Network Requests

The extension makes network requests only to these endpoints:

1. Online Picketline API

https://onlinepicketline.com/api/blocklist.json

Frequency: Every 15 minutes

Purpose: Keep labor action data current

Data Sent: API key, extension version, content hash (for caching)

Privacy: No user identification, no IP logging

2. GitHub Releases API

https://api.github.com/repos/online-picket-line/opl-browser-plugin/releases/latest

Frequency: Once every 24 hours

Purpose: Notify you when updates are available

Data Sent: None (public API)
